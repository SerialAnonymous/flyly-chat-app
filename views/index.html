flyly

flyly - a chat app

built on an unholy amalgamation of Node.js and socket.io and PHP

Built on nodejs

Node.js is a server side framework build on Chrome's V8 JavaScript Engine.
Is fast, built for real-time web apps and has evergrowing npm.
Way too many frameworks that are built on Node.js

Uses socket.io

A framework for real time web applications.
It enables realtime, bi-directional communication between web clients and servers.
Essentially acts like a websocket.
Has fallback patterns included.
Runs on Node

Responsive Design

Works perfectly well on mobile devices.
Home, login and signup pages scale with the help of bootstrap.
chat page does so without bootstrap.
Even has a different ui/ux for mobile devices!

Does code highlight

Just include three ticks ``` followed by language name before code and 3 ticks after code to see highlighted code!.
uses prism.js.
Supports most majpr languages.

MYSQL Database

The simplest and most common database to use.
We don't have any form of complex database thus the simple mysql.
Is fast for small databses like this.

Uses simple BootStrap for front-end

Bootstrap is a mobile-first free and open-source front-end web framework for designing websites and web applications.
Makes things look neat while also making it simpler for the developer.

Apache PHP node backend

Not exactly a beautiful amalgamation

Usage of git for project management

A defentie lot of git was learnt.
Uhg conflicts!

Includes multiple ajax patterns

Multi-stage download of profile pictures and feature list.
Predictive Fetch of older messages.
Delayed send in searching through contacts.
Fallback patterns through sockets.
